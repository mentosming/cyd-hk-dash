CYD-DASH

香港車載儀錶板。一塊 ~HK$80 嘅 ESP32 屏，插車 USB，即刻睇到過海隧道行車時間、現時時段收費、附近咪錶空位同油價。

過海隧道

紅隧 / 東隧 / 西隧 雙向實時行車時間，加現時時段收費同下次轉價倒數。

主要幹道

獅隧、大老山、青沙、屯赤、將隧⋯ 三條路線任你揀。

附近咪錶

撳一下，搵到 4 公里內有空位、而家准泊嘅街道。

油價

消委會五間油公司牌價，最平自動高亮。

⚡ 一鍵燒錄

插咗 ESP32 落 USB，撳個掣就搞掂。唔使裝任何軟件。

呢個瀏覽器唔支援 Web Serial。請用 Chrome、Edge 或 Firefox 151+（桌面版）。Safari 唔支援。 要喺 HTTPS 網頁先用得。

你需要咩

  1. ESP32-2432S028R（俗稱 Cheap Yellow Display / CYD，2.8" 320×240 觸控屏）—— 淘寶 / AliExpress 搜型號，約 HK$60–90。
    ⚠️ 要單 micro-USB 嘅 ILI9341 版本；雙 USB 嘅 v2/v3（ST7789）暫未支援。
  2. 一條 micro-USB 線（燒錄用）+ 車上 5V USB 供電
  3. 一部 iPhone（iOS 17+）裝 HK CarDash app

點裝

  1. 上面撳「燒錄 firmware」→ 揀 serial port → 等 30 秒
  2. iPhone 裝 HK CarDash（App Store 上架中）
  3. 屏幕會出一個 QR → App 撳「掃描配對 QR」→ 掃 → 完

之後每次上車自動連接，唔使再掃。顯示屏本身零上網 —— 手機負責攞數據同定位，經藍牙推細 payload 上屏。
